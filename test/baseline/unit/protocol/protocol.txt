VisIt protocol report
========================================================================
Command line: 
    /nfs/tmp2/brugger/test_trunk/src/exe/visitprotocol

VisIt contains state objects that are used to communicate data from
one VisIt component to another. These state objects are described by
XML files and the XML files are used to automatically generate source
code representing the state object. This tool prints out a description
of all of the state objects in use for a version of VisIt. The output
of this program can be used to determine whether 2 versions of VisIt
have the same state object definitions, thus having the same binary
protocol for communication. You can redirect the output of this program
to a file on the 2 systems being compared and then you can use a text
differencing program to look for changes that make the protocols
incompatible.
