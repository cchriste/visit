VisIt 1.2.7 Release Notes

Welcome to VisIt's release notes page. This page describes the important enhancements and bug-fixes that were added to this release.

Bugs fixed in version 1.2.7

  • VisIt issues a warning when the size of history variables is not the expected value.
  • VisIt uses invalid extents when using material selection.
  • VisIt does not handle quad cells that are stored in the file as hexes.
  • Extreme slowness when opening the first file.
  • The metadata server should load plugins on startup.
  • Report domain when internal error with variable size.
  • PickByNode and PickByNode queries need range checking so VisIt does not crash.
  • Magnitude of contour plot with log scaling acted funny for an expression.
  • When the VisIt is run in parallel, the vis window has a different view when it contains curves than it would had VisIt been run serially.
