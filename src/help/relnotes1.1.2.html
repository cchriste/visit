VisIt 1.1.2 Release Notes

Welcome to VisIt's release notes page. This page describes the important enhancements and bug-fixes that were added to this release.

Features added in version 1.1.2

  • VisIt's viewer now has plot and operator toolbars that allow you to easily create plots and add operators.
  • VisIt now has a database reader plugin that reads point mesh and data stored in ascii text files.
  • Reference lines can now update curve plots when the active database timestep changes.
  • VisIt's Reflect operator window has been redesigned so that you can choose the reflection octants using a 3D widget.
  • Reference lines can now extract data and generate curve plots for multiple variables.
  • VisIt's GUI has a new preferences window that allows you to set a flag that governs whether window attributes are copied to new vis windows when they are referenced for the first time.

Bugs fixed in version 1.1.2

  • VisIt's parallel compute engine crashes if it is started by reopening a file.
  • The viewer doesn't ignore interactive tools if the compute engine is busy.
  • Pick does not report mixed variable values for cells that have mixed materials.
  • The variable legend sometimes uses the wrong colors when the plot contains only constant values.
  • VisIt's KullLite database reader plugin does not handle wedges correctly.
  • The GUI cores on exit when it is built with g++ 3.x.
