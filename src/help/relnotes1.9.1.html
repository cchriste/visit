VisIt 1.9.1 Release Notes

Welcome to VisIt's release notes page. Release 1.9.1 is primarily a bug fix release to release 1.9. Look here to see a complete list of bugs fixed in release 1.9.

Bugs fixed in version 1.9.1

  • Fixed path-parsing problem on Windows systems that prevented plugins from loading on diskless configurations.
  • VisIt fails to locate and use movie templates when using the Save Movie Wizard.
  • The Spreadsheet plot is missing its top level menu in the MacOS X of VisIt.
  • The XDMF reader crashes the compute engine when reading data that uses the VxVyVz coordinate format.
  • VisIt does not use the compute engine configuration saved a session file when using a session file to generate a movie with "visit -movie".
  • Clicking on an imported color table in the Color Table window causes VisIt's GUI to crash.
  • VisIt truncates the username on Windows platform when 'Change username' option is chosen while connecting to remote machine.
