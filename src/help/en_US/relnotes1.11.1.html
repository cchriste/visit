VisIt 1.11.1 Release Notes

Welcome to VisIt's release notes page. Release 1.11.1 is primarily a bug fix release to release 1.11. Look here to see a complete list of bugs fixed in release 1.11.

Bugs fixed in version 1.11.1

  • The host profiles for LLNL's juno system have been corrected to handle node names with 4 digits.
  • The source code was modified so that VisIt would compile with the gcc 4.3.x compilers.
