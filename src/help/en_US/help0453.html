Keyframing Window

Keyframe animations are designed using VisIt's Keyframing Window (see figure), which you can open by selecting the Keyframing option from the Main Window's Controls menu. The window is dominated by the Keyframe area, which consists of many vertical lines that correspond to each frame in the animation and horizontal lines, or Keyframe lines, that correspond to the state attributes that are being keyframed. The horizontal lines are the most important because they allow you to move and delete keyframes and set the plot range, which is the set of animation frames over which the plot is defined.
