VisIt 1.12.1 Release Notes

Welcome to VisIt's release notes page. Release 1.12.1 is primarily a bug fix release to release 1.12. Look here to see a complete list of bugs fixed in release 1.12.

Bugs fixed in version 1.12.1

  • The ray-casted volume renderer no longer gets lighting queues wrong when the opacity variable is different than the color variable.
