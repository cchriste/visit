VisIt 2.5.2 Release Notes

Welcome to VisIt's release notes page. Release 2.5.2 is primarily a bug fix release to release 2.5.1. It fixes additional bugs in release 2.5.1. Look here to see a complete list of changes in release 2.5.1.

Configuration changes in version 2.5.2

  • The host profiles for LLNL's atlas machine were removed due to machine retirement.
  • The partition in the host profile for ORNL's lens machine was changed from analysis to computation.

Bugs fixed in version 2.5.2

  • Plots using variables created with the DataBinning operator would delete their actors when changing active plots in the plot list. This caused the plot to appear in the Pending state and lead to unnecessary re-execution of the plot.
  • Clearing pick points turns off the Spreadsheet plot's tracer plane.

Enhancements in version 2.5.2

  • Console output from remote mdserver or compute engines is now routed to VisIt's Output window. This ensures that the output is available for platforms such as MacOS X and Windows where the VisIt application does not use a console. Console output from those programs sometimes indicates the cause of client/server launch failures.
