VisIt 1.10.1 Release Notes

Welcome to VisIt's release notes page. Release 1.10.1 is primarily a bug fix release to release 1.10. Look here to see a complete list of bugs fixed in release 1.10.

Bugs fixed in version 1.10.1

  • The Silo reader causes a crash when reading multimat objects whose material entries are all set to "EMPTY".
  • The InverseGhostZone operator now operates correctly on blocks that have no ghost zones.
  • Deleting an operator by clicking the red X in the plot list now works for plots that are not selected.
  • Fixed more cases of failure in the NumNodes query.
  • Fixed the usage of the loopback device (127.0.0.1) for Windows.
  • Fixed a small memory leak on Windows platfrom.
