Pseudocolor plot

The Pseudocolor plot, shown in figure, maps a scalar variable's data values to colors and uses the colors to "paint" values onto the variable's computational mesh. The result is a clear picture of the database geometry painted with variable values that have been mapped to colors. You might try this plot first when examining a scientific database for the first time since it reveals so much information about the plotted variable.
