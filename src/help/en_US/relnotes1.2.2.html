VisIt 1.2.2 Release Notes

Welcome to VisIt's release notes page. This page describes the important enhancements and bug-fixes that were added to this release.

Features added in version 1.2.2

  • VisIt has a new program to convert databases to databases of different formats. The tool, which can be run using: visit -convert, can currently convert any of VisIt's supported data file formats into Silo or VTK files.
  • VisIt has a new material interface reconstruction algorithm that is 3 to 6 times faster than the old algorithm. The new algorithm uses less memory, produces fewer cells, and always has perfect connectivity. Memory usage is lessened and speed is increased for all areas of VisIt's execution pipeline.
  • Windows can now popup posted if you click the "Post windows by default" option in the Preferences window.
  • The Mesh plot now goes opaque when appropriate.
  • The Pick window has a new toggle that allows it to keep its pick points even when they are cleared from the vis window.
  • The Pick window has been changed so that multiple lines of text can be selected and pasted into other applications.
  • The Vector plot window has been reorganized a little to show which options are related.
  • VisIt's CLI has new methods to disable to the toolbar.
  • "Reset view" and "Recenter view" buttons have been added to the View window's Advanced tab.
  • VisIt's CLI has been enhanced so it is not so picky about taking colors that contain floating point numbers.

  • The view settings have been completely separated into 2D, 3D and Curve views throughout all of VisIt. This has resulted in a number of changes in the GUI, the Python client and the Java client.

    In the GUI, the view window has an additional panel with curve view attributes. They consist of the viewport, domain and range.

    In the Python interface, Get/SetViewCurve, ViewCurveAttributes, View2DAttributes and View3DAttributes have been added. The ViewAttributes class is still there for backward compatability, but will be removed in release 1.3. ViewCurveAttributes only has curve attributes, View2DAttributes only has 2D attributes and View3DAttributes only has 3D attributes (as opposed to the ViewAttributes which has them all combined.) GetView2D was modified to return a View2DAttributes object and GetView3D was modified to return a View3DAttributes object. SetView2D takes both View2DAttributes and ViewAttributes objects and SetView3D takes both View3DAttributes and ViewAttributes objects (again this was done to help with backward compatibility.)

    In the Java interface, Get/SetViewCurve, ViewCurveAttributes, View2DAttributes and View3DAttributes have been added. ViewAttributes has been removed, which will cause some backward compatability problems.

Bugs fixed in version 1.2.2

  • Lineout with endpoints that do not intersect a plot cause VisIt to crash.
  • The ParallelMerge method has been made more flexible so VisIt can tolerate more variation in data among processors.
  • The compute engine crashes when index-selecting out of range in a FilledBoundary plot.
  • VisIt can output inconsistent pick results.
  • Picking on a Pseudocolor plot with materials as a pick variable gives a bad pick result.
  • Configuration files can specify two default hosts for a set of host profiles.
  • The GUI and the CLI do not honor the saved Plot and Operator attributes in the config file.
  • Resetting the Plot options for a Subset plot of materials can result in a plot with zero colors.
  • Extra lines in sliced wireframe subset plot of materials.
  • Setting rendering options no longer takes place immediately.
  • Changing time steps with a virtual database removes the time step highlight.
  • The Replace button does not work with time-varying databases.
  • Spin mode should suspend when the vis window's popup menu is active.
  • The order of "." and ".." in the file list view has changed.
  • The mdserver should print exceptions that make it to the top level in the log file.
  • Picking on the mesh plot returns an unneeded field.
  • Lineout is missing in the Query window.
  • The mouse cursor stays a stopwatch when the viewer can't create a plot.
  • Windows that remember their old positions can popup off screen.
  • Problem with creating plots that want both Scalar and Material variables.
  • Moving the time slider while an animation is playing does not work.
  • It can be difficult to highlight files in the GUI.
  • Material interface reconstruction uses a lot of memory.
  • Using the plot turndown arrow causes the GUI's Hide/Show buttons to be disabled.
  • Make "Select by" options in the Contour Plot window plural.
  • The Reflect operator window does not show the axis labels for the back view.
  • The Threshold operator window does not reset properly.
  • The curve view is not set properly after doing a lineout.
  • Changing a curve attribute causes the curve view to be reset.
  • Curve plots ignore the limits in the view window.
  • Pan and zoom in the 3D view attributes are not interpolated by EvalCubicSpline.
