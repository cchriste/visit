Operator options


Figure 23

The Operator menu and toolbar allow you to add new operators and remove operators from plots. The Operator menu is always available in the Popup menu but the Operator toolbar is not visible by default. If you want to make the Operator toolbar visible, you can turn it on in the Popup menu's Customize menu. The Operator menu and Operator toolbar are shown in figure and figure.
