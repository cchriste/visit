Plugin Manager Window

The Plugin Manager Window, shown in figure, allows you to see which plugins are available for plots, operators, and databases. Not all plugins have to be loaded, in fact, many operator plugins are not loaded by default. The Plugin Manager Window allows you to specify which plugins are loaded when VisIt is started. The Plugin Manager Window is activated from the Main Window's Options menu after clicking on the Plugin Manager option.
