Posting a window

Windows that have a Post button can be posted to the Main Window's Notepad area. Clicking on a window's Post button hides the window and adds its controls to a new tab in the Notepad area. Posting windows allows you to have several windows active at the same time without cluttering the screen. When a window is posted, its Post button turns to an UnPost button that, when clicked, removes the posted window from the Notepad area and displays the window in its own window. figure shows an example of a window with a Post button and also shows the same window when it is posted to the Notepad area.
