VisIt 1.0.3 Release Notes

Welcome to VisIt's release notes page. This page describes the important enhancements and bug-fixes that were added to this release.

Features added in version 1.0.3

  • VisIt now supports database plugins.
  • Material interface reconstruction is faster.
  • The Subset plot can now show groups.
  • VisIt now provides a query for volume of a material.
  • VisIt now has a file information window that provides information about the open file.
  • The current working directory is available as an option in the File Selection window.
  • The Contour plot now has an opacity slider for the single color option.
  • VisIt now has a '-dir' command line option that allows the location of VisIt to be specified on remote machines.
  • The password window now shows the username when it is appropriate.
  • The Silo database plugin can read in legacy expression definitions.

Bugs fixed in version 1.0.3

  • Reflected Subset plots caused the engine to crash.
  • The test suite crashes VisIt.
  • Subsetting a Pseudocolor plot can crash the engine.
  • The Threshold operator gives a confusing error message when an invalid variable is used.
  • The GUI hangs when changing files too quickly.
  • Lighting causes 2D plots to be too dark.
  • The Windows version of VisIt cannot connect to remote machines.
  • Object lights are not working correctly.
  • Cannot do transparent and thick lines at the same time.
  • Ambient light intensity does not affect Pseudocolor plots.
  • Confusing error message with BadIndexException.
  • Recent paths that have spaces are not read back in properly.
  • Subset plots of materials in subdirectories cause an extra popup menu in the variable menu.
  • The test results Web page should sort past results from newest to oldest.
  • The VisIt User's Manual needs to be updated.
  • Help generation tools don't work with FrameMaker 7.0.
  • Config files on Windows platform are not saved in the right place.
  • Subset plot gives confusing error message when there is no data.
  • Pieces of interactive tools are obscured by transparent plots.
  • Annotations for thin meshes are badly skewed.
  • Bounding box navigation mode generated two redraws with transparency enabled.
  • 2D vector defvars don't show up for point2d.silo.
  • Problems with mesh quality metrics and reduced connectivity points.
