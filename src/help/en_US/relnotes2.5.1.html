VisIt 2.5.1 Release Notes

Welcome to VisIt's release notes page. Release 2.5.1 is primarily a bug fix release to release 2.5.0. It fixes additional bugs in release 2.5.0. Look here to see a complete list of changes in release 2.5.0.

Bugs fixed in version 2.5.1

  • The XRayImage query now correctly handles 3D rectilinear meshes.

Enhancements in version 2.5.1

  • Paul Stewart contributed the BATL reader to visualize BATS-R-US/CRASH code output.
