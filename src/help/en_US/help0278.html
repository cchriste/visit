Switching window modes


Figure 9


Figure 10

The Popup menu contains a Mode menu (see figure) that contains the 5 window modes. You can select a window mode from the Mode menu to change the vis window's mode. If you want to move or zoom the plot, choose navigate or zoom modes. If you want to extract data from the plots in the vis window, choose lineout mode or one of the pick modes. You can also use the Mode toolbar to change the vis window's window mode.
