File Selection Window


Figure 1

The File Selection Window, shown in figure, allows you to select files and simulations by browsing file systems either on your local computer or the remote computer of your choice. You can open the File Selection Window by choosing the Select Files option from the Main Window's File menu. When the window opens, its current directory is set to the current working directory or a directory from VisIt's preferences.
