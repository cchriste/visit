VisIt 1.3b1 Release Notes

Welcome to VisIt's release notes page. This page describes the important enhancements and bug-fixes that were added to this release.

Features added in version 1.3b1

  • Build SAMRAI on gps and mcr.
  • Would like VisIt to recognize "mkf" for Master Kull File.
  • Timeout not exposed on MCR.
  • Should load database plugins on demand in engine.
  • Put database reader libraries that we build ourselves into the third_party directory in VOB
  • Remove engine dependence on Qt.
  • The icons in the tool bar are difficult to see.
  • Should issue an informational message when loading plugins

Bugs fixed in version 1.3b1

  • pick on non-plotted, expression variable with #doms < #procs causes hang
  • Viewer hung after mdserver not accessed for a while.
  • Even though SR is always, huge dataset gets transferred to viewer.
  • Area between 2 curves query crashes viewer if the curves are not from the same database
  • zero entry in species speclist causes crash during query
  • Problems with Queries->Zone Pick
  • Viewer crashes when SR image does not come back correctly.
  • LD_LIBRARY_PATH should be set on Suns.
  • Variables drop down contains wrong variable list.
  • make LEOS plugin generate expression friendly variable names
  • The viewer kept trying to restart the compute engine even after the GUI told the viewer to quit.
  • PDB files are opened as Silo files if you've ever opened a Silo file.
  • Project to 2D not saved/restored correctly
  • Vector plot displays NaNs for legend in SR mode
  • Does sending a ClearCache RPC to the engine really free memory?
  • Plot has wrong view when Scalable Rendering kicks in after other plots are already drawn.
  • Changing #samples in Volume plot when autoupdate is on can hang the viewer.
  • Vector point vars not read correctly.
  • Wild card matching failed.
  • VisIt can't read old style silo files
  • Coloring with wrong extents with point meshes.
  • Range of vector is being determined using only what has been displayed.
  • Confusion over centering with complicated case.
  • Confusion over expanded operators.
  • xml2atts writes out the list of enums for all xml files that it has seen
  • VisIt can't plot mesh for VTK file.
  • pdb tests trigger bad material number warning
  • Problem with database code generation.
  • Engine relaunching has odd behavior.
  • Some zonal expressions are identified as nodal.
  • Visit no longer builds on Hyper
  • Failure to launch engine can result in infinite restarts.
