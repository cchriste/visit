VisIt 2.13.1 Release Notes

Welcome to VisIt's release notes page. This page describes the important enhancements and bug-fixes that were added to this release.

Sections

  • Bug Fixes
  • Enhancements
  • Changes for VisIt developers

Bugs fixed in version 2.13.1

  • Modified the query over time to store the results in double precision instead of single precision.
  • Modified the VTK Reader to ensure material scalars in multi-block datasets are read correctly.
  • For Windows platform, VisIt will now display an error message if VISITSSH defined, but does not point to an executable, and if a parallel engine is requested but mpiexec.exe cannot be found.

Enhancements in version 2.13.1

  • Enhanced the SPCTH reader to process a list of MTMD files as one MTMD file.

Changes for VisIt developers in version 2.13.1

  • Modified build_visit to pass the zlib directory information to the configure command for hdf5 when -zlib was specified on the command line.
  • Modified bv_pyside.sh to patch pyside2 that removes the conditional collecting of the Multimedia, WebEngineWidget, and WebKit modules.
  • Fixed the QWT make install symlink error for OSX.
