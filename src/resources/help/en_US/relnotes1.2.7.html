VisIt 1.2.7 Release Notes

Welcome to VisIt's release notes page. This page describes the important enhancements and bug-fixes that were added to this release.

Features added in version 1.2.7

  • The iso volume operator has been added to VisIt.
  • The performance of the material interface reconstruction, contour and slice operators has been improved by about 5 - 10 percent.
  • The reading of unstructured meshes from Silo files has been improved by about 10 percent.

Bugs fixed in version 1.2.7

  • The values for eff_stress from Mili files are incorrect.
  • VisIt should not issue a warning message when the number of entries in a zonal variable equals the number of entries in the mesh excluding ghost zones.
  • VisIt uses invalid variable extents when material selecting a variable where the number of entries in a zonal variable equals the number of entries in the mesh excluding ghost zones.
  • VisIt does not handle quad cells that are stored in the file as hexes.
  • VisIt is slow opening a Mili file.
  • VisIt should report the domain number when reporting an error in a data file.
  • 2D vectors should be drawn on top of the mesh to improve the visibility of the vectors.
  • VisIt crashes when PickByNode and PickByNode queries are given out of range index values.
  • The contour levels are incorrect when using log scaling on expression variables.
  • VisIt sets the window limits incorrectly for curve plots when run in parallel.
  • Creating a boundary plot of an index selected variable doesn't work.
