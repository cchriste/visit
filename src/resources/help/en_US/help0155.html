Operator Types

VisIt is installed with operator plugins, which perform a wide variety of functions. Some of the operators are not be enabled by default so they do not show up in the Operator menu. Use the Plugin Manager Window, which can be opened by clicking on the Plugin Manager option in the Main Window's Preferences menu, to enable additional operators or disable operators that you rarely use. This section explains each operator in alphabetical order; not in terms of an operator's importance.
