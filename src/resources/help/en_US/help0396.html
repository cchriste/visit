Setting background and foreground colors


Figure 8


Figure 9

The Colors tab has two color buttons that allow you to set the background and foreground colors. To set the background or foreground color, click the Background or Foreground color button and select a color from the Popup color menu. Releasing the mouse outside of the Popup color menu (see figure) cancels color selection and the color is not changed. Once you select a new color and click the Apply button, the colors for the active visualization window change. Note that each visualization window can have different background and foreground colors.
