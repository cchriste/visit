VisIt 2.6.2 Release Notes

Welcome to VisIt's release notes page. Release 2.6.2 is primarily a bug fix release to release 2.6. It fixes additional bugs in release 2.6. Look here to see a complete list of changes in release 2.6.

Configuration changes in version 2.6.2

Bugs fixed in version 2.6.2

  • A memory error was corrected in the Silo database reader that sometimes caused plots that involved materials to fail.
  • Saved python expression scripts can now be successfully found and loaded on Windows.
