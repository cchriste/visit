VisIt 0.9.12 Release Notes

Welcome to VisIt's release notes page. This page describes the important enhancements and bug-fixes that were added to this release.

Features added in version 0.9.12

  • VisIt has been ported to the MS Windows operating system.
  • VisIt now supports variable expressions.
  • VisIt now has documentation for the VisIt Python Interface.
  • Visualization window attributes such as the view can now be copied to other visualization windows.
  • Mesh plots can now use OpenGL display lists and are now much faster.
  • Pick can now report on material variables.
  • VisIt supports groups.
  • VisIt supports point meshes.
  • VisIt has a new cone operator that can intersect plots and optionally project them to a flat surface.
  • VisIt has a new index select operator.

Bugs fixed in version 0.9.12

  • Picking on sliced 3D plots gives bad pick information.
  • Subset plots of domains are broken.
  • On Linux, applying transform to 2d plots causes VisIt to hang.
  • The onion peel operator can malfunction when auto-update is turned on.
  • Visit CLI does not launch on benares, a red hat 7.1 laptop
  • Doing a lineout on an expression variable crashes the viewer.
  • Viewer crashes when performing a lineout when plots are hidden.
