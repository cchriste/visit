Decimate operator


Figure 22


Figure 23

The Decimate operator, shown in figure, removes nodes and cells from an input mesh, reducing the cell count while trying to maintain the overall shape of the original mesh. The Decimate operator can currently operate only on the external surfaces of the input geometry. This means that in order to apply the Decimate operator, you must first apply the ExternalSurfaces operator, which will be covered later in this chapter.
