Setting the cylinder's endpoints

There are two ways to set the endpoints for the Cylinder operator. First of all, you can open the Cylinder operator attributes window (see figure)and type new 3D points into the Endpoint 1 and Endpoint 2 text fields. The second, and more interactive way to set the endpoints for the Cylinder operator is to use VisIt's interactive Line tool, which is discussed in the Interactive Tools chapter. The Line tool lets you interactively place the Cylinder operator's endpoints anywhere in the visualization. The Line tool's endpoints correspond to the centers of the cylinder's top and bottom circular faces.
