Color table window


Figure 21

You can open VisIt's Color table window, shown in figure, by selecting the Color table option from the Main Window's Controls menu. The Color table window is vertically separated into two main areas. The top area, or manager portion of the window, allows you to set the active color table as well as create or delete new color tables. The bottom area, or editor portion of the window, allows you to edit color tables by adding, removing, moving, or changing the color of color control points. A color control point is a point with a color that influences how the color table will look.
