Setting the histogram data range


Figure 17

By default, the Histogram plot profiles a variables entire data range. If you want to restrict the Histogram plot so it only takes a subset of a variable's data range into consideration when assigning values to bins, you can set the minimum and maximum values that will be considered by the Histogram plot. To specify a data range, click the Specify Range check box and then type in floating point numeric values into the Minimum and Maximum text fields in the Histogram plot attributes window (see figure) before clicking its Apply button. Once the data range is set, the Histogram plot will restrict the values that it considers to the specified data range.
