VisIt 1.0.4 Release Notes

Welcome to VisIt's release notes page. This page describes the important enhancements and bug-fixes that were added to this release.

Features added in version 1.0.4

  • VisIt now has a Query window under its Controls menu that allows the user to query values from the open database as well as retrieve information such as total volume or surface area.
  • VisIt now provides a Renderer options window under the Options menu that allows the user to specify stereo rendering, antialiasing, surface representation, and use of OpenGL display lists as well as view the frame rate, extents, and approximate number of triangles rendered.
  • VisIt's view window has been redesigned.
  • The Main Window's Window menu now has options for controlling the active vis window.
  • Less frequently used menu options in the vis window's popup menu have been relocated to the GUI.
  • VisIt now uses the labels stored in a files option list for the to label the vis window axes.
  • The viewer sometimes takes a long time to disappear after a long VisIt session.
  • Pick points now display the original location for point meshes.
  • The compute engine is not launched when an invalid file is opened.
  • Image differences in the test suite are brighter.
  • Would like '*' in the File Selection window to not match filenames that begin with dot.
  • The File Selection window no longer replaces replaces the path when it is invalid.
  • Host profiles are now used to populate the File Selection window's list of hosts.

Bugs fixed in version 1.0.4

  • Material selection crashes VisIt's engine on certain mesh types.
  • VisIt sometimes crashes when communicating ghost zones.
  • Number of 2D view axis labels/gridlines changes unexpectedly when panning.
  • Interactive view changes for non-active windows appear in the view window.
  • Subset plots of domains change colors in parallel.
  • Volume plots with 2D datasets runs out of memory.
  • VisIt's launch script assumes csh.
  • VisIt SAF database reader doesn't handle databases with multiple time states.
  • Sometimes the mdserver does not give an up directory option on the Windows platform.
  • Changing font size in the help window does not take effect immediately.
  • Bookmarks in the help window malfunction sometimes.
  • SAF does not compile.
