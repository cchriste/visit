VisIt 2.10.3 Release Notes

Welcome to VisIt's release notes page. This page describes the important enhancements and bug-fixes that were added to this release.

Sections

  • Bug Fixes
  • Enhancements
  • Changes for VisIt developers

Bugs fixed in version 2.10.3

  • Fixed a bug in the LAMMPS reader that caused axis scaling to always be applied.
  • Command recording for changing PickAttributes now reports the correct 'getter' method: GetPickAttributes.
  • Fixed a bug where VisIt could draw a Pseudocolor plot of a .vtu file with only points (no cells) defined, but could not draw a Mesh plot of the same dataset.
  • Fixed a few issues related to Pseudocolor plot's use of ColorTable opacity: viewer crash when creating a transparent colortable via cli; Pseudocolor plot affected by changes to a ColorTable it isn't utilizing; changing the ColorTable's opacities would make the Pseudocolor plot go blank.
  • Fixed a bug using system python on Windows, error parsing added arguments.
  • Fixed reader for EnSight gold files where geometry file name specified by wildcards and a '001' numbered geometry file did not exist.
  • Fixed a bug that prevented multigrid, Fortran binary PLOT3D files from being read successfully.
  • Fixed a bug that prevented multigrid, Fortran binary, double-precision, iblanked PLOT3D files from being read successfully.
  • Fixed a bug that caused invalid expression variables after deleting a plot and switching back to its active source.
  • Fixed a bug with pick on the reflected part of subset plots with reflected operator applied.

Enhancements in version 2.10.3

  • Enhanced the Resample Operator to handle polygon cells.
  • The Cosmos++ reader was enhanced to support reading particle data.
