VisIt 1.0.7 Release Notes

Welcome to VisIt's release notes page. This page describes the important enhancements and bug-fixes that were added to this release.

Features added in version 1.0.7

  • The Mesh, Pseudocolor, and Subset plots now have options for mesh smoothing. Mesh smoothing removes small points that are common after material interface reconstruction.
  • The Connection Progress window, which shows the status of compute engine launches, is now available on the AIX platform since it now works correctly with compute engines submitted to the batch system.
  • The Engine window now displays the number of processors and nodes used by a compute engine so it is immediately obvious if the compute engine is serial or parallel.
  • VisIt now provides ten different line thicknesses for plots that use lines.
  • VisIt can now save Postscript files.
  • The On/Off buttons in the Subset window have been renamed to All/None.

Bugs fixed in version 1.0.7

  • VisIt hangs or crashes when performing species selection.
  • Material selection sometimes produces results that look erroneous.
  • VisIt crashes when picking on a subset plot of materials containing mixed zones.
  • VisIt sometimes crashes while communicating ghost zones when running in parallel.
  • Pick returns invalid results with material selected plots.
  • Setting a SIL restriction for a plot in a window created by cloning causes VisIt to crash.
  • Slicing along the problem boundary results in a plot with no geometry.
  • The Subset plot does not show all domains after replacing a file.
  • The Surface plot fails to update when certain options are changed.
  • The Mesh plot shows lots of extra junk lines if material selection has been performed.
  • The Host Profile window resets values.
  • The Vector plot is missing a legend.
  • If the compute engine does not have the requested operator, it applies the wrong operator.
  • Having a "config" directory in your current directory crashes VisIt.
  • The Subset plot sometimes has a bad legend.
  • VisIt no longer recenters the view when reopening or replacing files unless the Autocenter view toggle is set in the View window.
  • When the Index Select operator is applied to a Mesh plot that has been thresholded, the resulting plot is incorrect.
  • The Subset plot's legend does not preserve the order of materials in the database.
  • Setting the "Select by" method in the Contour plot window now works correctly with auto update.
  • Adding -L/usr/local/lib causes problems in VisIt's link line.
