Surface color


Figure 49

By default, the Surface plot is colored by the variable value or Z-value as with the Pseudocolor plot. The second coloration scheme uses constant coloration where the entire surface is the same color (see figure). To choose a coloration scheme, click on the Constant or Z-Value radio buttons. When the plot uses constant coloring, you can change the color by clicking on the color button next to the Constant radio button and selecting a new color from the popup color menu. When the plot uses coloring based on the Z-Value, you change colors by selecting a new color table name from the color table button next to the Z-Value radio button. The available color table names are an up-to-date list of VisIt color table names.
