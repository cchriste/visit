Locking views

The vis window can lock its view to other vis windows. When this toggle is set, making a change that affects the view in the active vis window will cause other vis windows that have the lock views toggle set to receive the same view as the active window. To lock the view, select the Lock view option from the Popup menu's View menu (see figure) or click on the Lock view icon in the Toolbar. Note that you can lock 2D and 3D windows separately.
