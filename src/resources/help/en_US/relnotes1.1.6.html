VisIt 1.1.6 Release Notes

Welcome to VisIt's release notes page. This page describes the important enhancements and bug-fixes that were added to this release.

Features added in version 1.1.6

  • VisIt has a new Histogram plot that can be used to show the distribution of values for a variable in a database.
  • VisIt has a new pick mode that allows you to pick on nodes of a mesh.
  • The vis window's popup menu has a new Clear submenu that lets you clear pick points, reference lines, and plots from the window.
  • You can now control the location of the major and minor tick marks for the 2d axes.
  • VisIt reads another PDB format.
  • VisIt now runs in parallel on LANL's Q machine.
  • Color tables can be saved to their own files, which can be shared among colleagues. The Color table window has a new "Export" button to export color tables to files.
  • The GUI's File menu has a new "Refresh files" button that refreshes the file list and allows VisIt to detect files that are new since the last time the directory was checked..
  • 2d zooming now has reference lines to make it more obvious where you are zooming.
  • VisIt can now save several views to the config file for use in later sessions. To save a view, click the "Save view" icon in the toolbar. Once several views are saved, you can click on their icons to choose between them. When "Saving settings", these views are saved to the config file so that they are available the next time you run VisIt.
  • The Shift key forces rubber band zooming to be square.
  • VisIt's Point tool has been improved.

Bugs fixed in version 1.1.6

  • VisIt's viewer crashes after slicing a mesh that has been onion-peeled.
  • VisIt crashes when doing animation due to a memory leak.
  • The Reset button causes the wrong plot attributes to be applied when some plugins are disabled.
  • The GUI hangs when the "Clone window on first reference" feature is enabled and no database was opened before changing the active window.
  • Cloning windows causes plots to be drawn immediately in the new window.
  • No ssh password prompt is available when -cli -nowin arguments are given.
  • The CLI has no way to return the coordinates for nodes.
  • Parallel compute engines are not clearing out of the VIEWS nodes.
  • Extra triangles in the middle of the dataset for a FilledBoundary plot of materials.
  • Zoom limits with zoom mode can't extend all the way to xmin, ymin.
  • Plugins are installed in /usr/gapps/visit by default.
  • The Reflect operator attributes window is not so clear for 2d.
  • Pick points disappear when a mesh plot is added.
  • The Area query returns negative values for "pixels".
  • Rubber band zooming where the new zoom rectangle has no area is no longer allowed.
