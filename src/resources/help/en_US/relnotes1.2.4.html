VisIt 1.2.4 Release Notes

Welcome to VisIt's release notes page. This page describes the important enhancements and bug-fixes that were added to this release.

Features added in version 1.2.4

  • VisIt now supports AMR meshes as first class mesh types, including reader plugins for SAMRAI and BoxLib (this feature was actually available in 1.2.3 but was not included in the release notes at that time.)
  • VisIt has a new Cylinder operator that slices data with a cylinder.
  • Directories that lack read permission are now grayed out in the File selection window.
  • Reopening a file at a different time state now causes a warning message because reopen should not be used to change active time states.
  • Mili files now return their times to the GUI so they can be displayed when a Mili database is expanded.
  • Plugins can now be off by default so they can be distributed but not necessarily loaded.
  • The xml2plugin tool now generates avtFileFormat code for database plugins.
  • The Mili database reader plugin now does less work on the metadata server so reading metadata is significantly faster.
  • VisIt's GUI has a new -sessionfile argument that lets you open a session when VisIt starts. On the Windows platform, double-clicking a session file opens the session file in the GUI and right clicking on a session file provides the option of using it to generate a movie.
  • The Annotation window now has a "No annotations" button that turns off all annotations.
  • The time in the vis window's user string updates when the plot list is modified.

Bugs fixed in version 1.2.4

  • expose/resize events of a window in scalable rendering mode causes exception then crash.
  • Scalable rendering does not reflect hide/show immediately.
  • "Using Scalable Rendering" label seems to always display "no".
  • Changing to scalable rendering draws un-drawn plot.
  • Scalable rendering mode even though 'never' was set.
  • Auto switch to scalable rendering causes full network re-execution.
  • "Auto" scalable rendering mode is too aggressive.
  • 2D plots and 1D curves do not have the right view when rendered using scalable rendering.
  • VisIt's compute engine crashes when you try to make a FilledBoundary plot of data from specmix_quad.silo.
  • NestedDomain ghosting object gets lost after time-step change.
  • VisIt crashes when given an expression that contains no database variables.
  • Kull Lite database reader plugin cannot support more than 20 open files.
  • Cannot do matvf expression for files where ghost zones are communicated.
  • The GUI gets virtual databases confused.
  • VisIt performs too many executions when replacing a database.
  • Pick does not work when an OnionPeel operator has been applied.
  • Reopening a file sometimes crashes the compute engine on AIX
  • Using "white" as the hostname does not pick up the current host profiles.
  • The compute engine is in a bad state after making a bad plot.
  • The if expression is making mistakes with inputs that are constants.
  • Slicing with plots using percent causes the plot to project to a bad view.
  • The GUI's file panel sometimes improperly expands MT databases when there are several open MT databases.
  • The compute engine crashes when the matvf expression is given invalid arguments.
  • The new rectilinear Subset plot drops out subsets in the presence of nested domains.
  • Moving the animation slider requires an execution even when the time state does not change.
  • Expressions from files are lost as you switch between databases.
  • The View window does not allow you to set small 2D view areas.
  • A session file with a Contour plot that uses NLevels to set the number of contours causes the compute engine to crash.
  • Rubberband zoom does not work in a vis window that contains curves.
  • The metadata server crashes when it cannot open a plugin.
  • Saved views are lost when saving a session file.
  • Fullframe mode does not always handle non-square windows.
  • All arrays are treated as floats in ExchangeScalar.
  • Histogram plot bars are not uniform in size.
  • Histogram plot limits are not correct.
  • Bounding box zoom does not work when fullframe mode is on.
  • The validVariable flag is not being honored for material variables.
  • Improve database library robustness with respect to empty file names.
  • VisIt transfers data from white to tidalwave very slowly.
  • The active window should not be saved in the settings; it should only be saved in a session file.
  • Setting the global opacity slider has no effect on a Subset plot that uses a single color.
